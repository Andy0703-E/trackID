Memulai

✅ API TrackID menggunakan sumber data publik dari ekspedisi terkait. Tidak diperlukan API key untuk penggunaan dasar. Rate limit berlaku untuk mencegah penyalahgunaan.

TrackID menggunakan dua sumber API utama: Biteship Public API untuk ekspedisi umum (JNE, J&T, SiCepat, dll.) dan SPX API untuk Shopee Express. Semua endpoint menggunakan HTTPS.

Base URL untuk Biteship Public:

URL
https://api.biteship.com/v1/public/trackings

Base URL untuk SPX:

URL
https://spx.co.id/shipment/order/open/order/get_order_info

Track Resi — Biteship

Endpoint ini digunakan untuk melacak resi dari ekspedisi yang didukung Biteship (JNE, J&T, SiCepat, TIKI, Ninja, dll.).

GET /v1/public/trackings/{awb}/couriers/{courier}

Lacak status pengiriman berdasarkan nomor resi dan kode ekspedisi.

ParameterTipeKeterangan
awb requiredstringNomor resi / AWB pengiriman
courier requiredstringKode ekspedisi (lihat daftar di bawah)

Header yang diperlukan:

HTTP Headers
Authorization: Public
Accept: application/json
Origin: https://biteship.com

Contoh permintaan dengan JavaScript:

JavaScript
const resi = 'JX3708794672';
const courier = 'jnt';
const url = `https://api.biteship.com/v1/public/trackings/${resi}/couriers/${courier}`;

const response = await fetch(url, {
  headers: {
    'Authorization': 'Public',
    'Accept': 'application/json',
    'Origin': 'https://biteship.com'
  }
});

const data = await response.json();
console.log(data);

Track Resi — SPX / Shopee Express

Endpoint khusus untuk melacak paket yang dikirim melalui Shopee Express (SPX).

GET /shipment/order/open/order/get_order_info
Query ParamTipeKeterangan
spx_tn requiredstringNomor resi SPX (format: SPXID...)
language_code optionalstringKode bahasa, gunakan id untuk Bahasa Indonesia
JavaScript
const resi = 'SPXID061577510985';
const url = `https://spx.co.id/shipment/order/open/order/get_order_info?spx_tn=${resi}&language_code=id`;

const response = await fetch(url, {
  headers: {
    'Accept': 'application/json',
    'Cookie': 'app_source=nss; app_lang=id',
    'Referer': `https://spx.co.id/m/tracking-detail/${resi}`
  }
});

const data = await response.json();
// Data riwayat ada di: data.data.sls_tracking_info.records
const records = data?.data?.sls_tracking_info?.records || [];

Error Codes

400
Parameter tidak valid atau tidak lengkap
404
Nomor resi tidak ditemukan
429
Rate limit tercapai, coba lagi nanti
500
Kesalahan server internal
503
API ekspedisi sedang tidak tersedia
