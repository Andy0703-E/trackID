Kebijakan Privasi

Kami menghormati privasi Anda. Halaman ini menjelaskan bagaimana TrackID mengumpulkan, menggunakan, dan melindungi informasi Anda.

Terakhir diperbarui: 1 Januari 2025

TrackID tidak memerlukan akun atau registrasi. Kami tidak menyimpan data pribadi Anda secara permanen. Nomor resi yang Anda masukkan hanya digunakan untuk menghubungi API ekspedisi terkait.

1. Informasi yang Kami Kumpulkan

TrackID dirancang dengan prinsip privacy by default. Kami hanya memproses data yang mutlak diperlukan untuk fungsi pelacakan:

  • Nomor Resi — digunakan sementara untuk menghubungi API ekspedisi. Tidak disimpan di server kami.
  • Pilihan Ekspedisi — menentukan API mana yang dihubungi. Tidak dicatat.
  • Data Teknis — browser type, IP address (melalui log server standar), dan waktu akses untuk keperluan keamanan dan pemantauan performa.

2. Bagaimana Kami Menggunakan Informasi

Informasi yang dikumpulkan digunakan semata-mata untuk:

  • Memproses permintaan pelacakan resi secara real-time
  • Memastikan ketersediaan dan performa layanan
  • Mendeteksi dan mencegah penyalahgunaan sistem
  • Menganalisis statistik penggunaan secara anonim untuk meningkatkan layanan

3. Cookie dan Penyimpanan Lokal

TrackID menggunakan cookie minimal yang diperlukan untuk fungsi teknis dasar. Kami tidak menggunakan cookie untuk pelacakan perilaku atau iklan. Pilihan ekspedisi terakhir mungkin disimpan di localStorage browser Anda untuk kenyamanan — data ini sepenuhnya lokal dan tidak dikirim ke server kami.

4. Pihak Ketiga

Saat Anda melacak resi, permintaan diteruskan ke API resmi ekspedisi terkait. Data yang dikembalikan oleh ekspedisi tunduk pada kebijakan privasi masing-masing ekspedisi tersebut. TrackID tidak bertanggung jawab atas praktik privasi pihak ketiga ini.

Kami juga menggunakan layanan CORS proxy publik sebagai perantara teknis. Proxy ini tidak menyimpan konten permintaan Anda.

5. Keamanan Data

Kami menerapkan langkah-langkah keamanan teknis yang wajar untuk melindungi informasi yang diproses melalui platform kami, termasuk koneksi HTTPS terenkripsi untuk semua komunikasi data.

6. Hak Anda

Karena TrackID tidak menyimpan data pribadi Anda secara permanen, tidak ada tindakan penghapusan yang diperlukan. Jika Anda memiliki kekhawatiran tentang privasi, silakan hubungi kami melalui email yang tersedia di halaman Tentang Kami.

7. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini sewaktu-waktu. Perubahan signifikan akan diinformasikan melalui pemberitahuan di halaman utama. Dengan terus menggunakan layanan TrackID, Anda menyetujui kebijakan yang berlaku.
